# CloudAILab learning index

CloudAILab is a sandbox with self-guided learning paths, not a classroom or LMS. Lessons organize existing test-backed scenarios and workflows without introducing separate authorization, grading, user accounts, certificates, or instructor administration.

- [Learning contract](learning-contract.md)
- [Identity and agent foundations path](identity-agent-foundations.md)
- [Data-only scenario starter](../../examples/scenario-starter/README.md)
- [One-time adaptation provenance](adaptation-provenance.md)
- [Machine-readable catalog](../../learning/catalog.json)
- [Catalog schema](../../schemas/learning/v1alpha1/learning-catalog.json)

Deterministic `cailab verify` and agent evidence remain authoritative. Completion checklists and reflection prompts help the learner explain the work but cannot change a verification result.
